# 天気予報スキル

## 概要

このスキルは天気予報データの取得リクエストを処理します。

## 処理フロー（スキル → サブエージェント → API）

```
1. [スキル] ユーザーから天気予報のリクエストを受け取る
       ↓
2. [サブエージェント] weather-agent を起動
       ↓
3. [API] WebFetch で気象庁APIから天気データを取得
       ↓
4. 取得結果をユーザーに返す
```

## サブエージェントの呼び出し

天気情報の取得が必要な場合は、**必ず Task ツールを使用して `weather-agent` サブエージェントを起動**してください。

```
Task ツールを以下のように呼び出す:
- subagent_type: "get-weather-4:weather-agent"
- prompt: "{地域名}の天気予報を取得してください"
- description: "{地域名}の天気取得"
```

weather-agent は WebFetch ツールで気象庁APIから天気データを取得します。

## 対応リクエスト例

- 「東京の天気を教えて」→ weather-agent を起動 → API呼び出し
- 「大阪の天気は？」→ weather-agent を起動 → API呼び出し
- 「札幌の天気」→ weather-agent を起動 → API呼び出し
